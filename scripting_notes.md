# Linux Basics

## Building Docker Images

- **Give all access to file/folder:**  
  `sudo chmod -R 777 data_postgres`
- **View History:**  
  `history`
- **Check OS Details:**  
  `cat /etc/os-release`
- **view all environment variable:**
  `env`
- **view environment variable with specific name:**
  `env | grep search_value`
